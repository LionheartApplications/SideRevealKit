Class Definition
  • SideRevealViewController

    UIViewController that is handling the reveal of view controller behind another one.

    That is the main class in SideRevealKit. You can use it from storyboards or from code.

    If you are using it from storyboards, you should set ‘Relationship Segue’s to your front and side controllers for easier setup. Have a look at SideRevealSegue and FrontRevealSegue. for more info.
